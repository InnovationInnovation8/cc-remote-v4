# CC Remote — セットアップガイド（Claude Code 向け）

## あなた（Claude Code）へ

このファイルを読んでいるなら、ユーザーは CC Remote のセットアップを依頼しています。
以下の手順に従って、自動でセットアップを完了してください。

## CC Remote とは

スマホのブラウザから、この PC の Claude Code セッションをリモート操作できるツールです。

## 前提条件

- `CC Remote.exe` がこのフォルダに存在すること
- ユーザーのスマホで CC Remote の PWA にアクセス済みであること

## セットアップ手順

### Step 1: ユーザーにスマホでの準備を依頼

ユーザーに以下を伝えてください:

「スマホのブラウザで https://cc-remote-api-701345803309.asia-northeast1.run.app を開いて、
Googleアカウントでログインしてください。ログインしたら『PC追加』ボタンを押して、
表示される6桁のペアリングコードを教えてください。」

### Step 2: ペアリングコードを受け取る

ユーザーから6桁の数字を受け取ったら、次のステップに進みます。

### Step 3: exe を CLI モードで起動

以下のコマンドを実行してください。
- `<PC名>` はユーザーに分かりやすい名前（例: "会社PC", "自宅デスクトップ"）に置き換え
- `<6桁>` はユーザーから受け取ったペアリングコードに置き換え

```powershell
& ".\CC Remote.exe" --pc-name "<PC名>" --pairing-code "<6桁>" --auto-startup
```

### Step 4: 結果を確認

- 「登録完了」と表示されれば成功
- セットアップ後、`CC Remote.exe` が常駐サーバーとして起動します
- PC再起動後も自動で起動します（--auto-startup を指定した場合）

### Step 5: ユーザーにスマホでの確認を依頼

「スマホの CC Remote を開いて（リロードして）、PC一覧にこのPCが表示されるか確認してください。」

## 再セットアップが必要な場合

既にセットアップ済みの状態で再実行するには `--force` フラグを追加:

```powershell
& ".\CC Remote.exe" --pc-name "<PC名>" --pairing-code "<6桁>" --auto-startup --force
```

## トラブルシューティング

- **「登録に失敗しました」** → インターネット接続を確認。スマホ側でペアリングコードを再発行して再試行
- **「無効なコード」** → ペアリングコードの有効期限は5分。期限切れならスマホで再発行
- **exe が起動しない** → Windows Defender の警告が出たら「詳細情報」→「実行」を選択
- **上記コマンドが失敗した場合** → `CC Remote.exe` をダブルクリックして起動し、画面の指示に従って手動セットアップしてください。詳細は `README.txt` を参照
