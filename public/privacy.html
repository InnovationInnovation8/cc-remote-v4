プライバシーポリシー

CC Remote v3  |  最終更新: 2025年4月

収集する情報

本ソフトウェアが取り扱うデータは以下のとおりです。

  • PINコード ハッシュ化して保存 — 認証に使用します。平文では保存されません。
  • セッションデータ ローカル保存 — ターミナルセッションの履歴・出力。利用者のPCのローカルDBにのみ保存されます。
  • FCMトークン ローカル保存 — プッシュ通知の送受信に使用します。Googleのサーバーを経由しますが、通知の内容（エラー発生など）は最小限です。
  • WebAuthn認証情報 ローカル保存 — 生体認証を使用する場合、認証情報IDと公開鍵のみをローカルDBに保存します。
  • 設定情報 ローカル保存 — フォントサイズ・テーマ・ショートカットなどの設定はブラウザのLocalStorageとローカルDBに保存されます。

データの保存場所

上記のデータはすべて利用者のPC上（ローカル）に保存されます。クラウドサーバーやデータセンターへのアップロードは行いません。

第三者提供

本ソフトウェアは収集した個人情報を第三者に販売・提供・共有しません。

ただし以下の外部サービスと通信する場合があります。

  • Google Firebase Cloud Messaging (FCM) — プッシュ通知の配信に使用（オプション機能）
  • Cloudflare Tunnel — 外部からのアクセスを中継するトンネル（オプション機能）
  • Anthropic Claude API — Claude Codeを通じたAI処理（利用者のAPIキーを使用）

これらのサービスの利用規約・プライバシーポリシーは各社のものが適用されます。

データの削除

ローカルDBファイル（cc-remote.db）を削除することで、保存されたすべてのデータを削除できます。ブラウザのLocalStorageはブラウザ側でクリアしてください。
